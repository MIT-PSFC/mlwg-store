# EBM tutorial

## Goal
The primary goals of this tutorial are to give a gentle introduction to generative modeling and energy-based models (EBMs) in particular. EBMs use concepts from physics, statistics, and machine learning that are interesting and useful even if you never use this class of model in the future. Try to work out the parts to fill in by yourself, but if you get stuck for more than a few minutes, please look at the solutions!

## Dependencies 
numpy, pytorch, matplotlib, tqdm

## File descriptions
- `EBM-tutorial-2026.pdf`: slides
- `ebm_tutorial.ipynb`: primary tutorial file containing sections to fill in 
- `ebm_tutorial_solutions.ipynb`: tutorial file with solutions
- `data.py`: contains routines for creating synthetic training data
